# Assets - Genesis Engine V2

This directory contains all multimedia resources necessary for **Genesis Engine V2** to function correctly.

**Genesis Engine V2** is an engine derived from [Friday Night Funkin'](https://github.com/FunkinCrew/Funkin), optimized to provide better performance and an improved gaming experience.
